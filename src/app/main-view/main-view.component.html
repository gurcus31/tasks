Simple Angular App

This is a simple Angular project, that consume resources from a drupal site.

From the left menu you can see all the tasks. This work with the GET METHOD and is calling a simple REST view from the drupal site.

Then you have a form for add new tasks. It use the PUT METHOD, if you inspect the element, you can see in the console the call of the /entity/node

Also, if you access to any of the task from the menu it will send you to the full view of the node. Inside it, you can go to Edit your tasks and made any change, this work with the PATCH METHOD, go to the console to see the call.
